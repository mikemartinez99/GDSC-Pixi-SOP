**Purpose:** 
To provide a standardized framework for implementing and managing software environments for core projects using Pixi, a versatile package manager that facilitates reproducibility across machines.

**Scope:** 
This SOP applies to any projects where a reproducible environment is needed, including R, Python, command-line tools, and mixed-language workflows.
